# HiPro Direct 応募手順

## サイト情報
- URL: https://talent.direct.hipro-job.jp/talent/for-local/top/
- ログイン: ブラウザで事前にログイン済みであること
- 特徴: 大手企業・地方企業の副業・兼業プロジェクト。ハイクラス人材向け

---

## 検索手順

`config.json` の `search_filters` を参照して以下の手順で操作する。

### 1. 案件一覧ページにアクセス
https://talent.direct.hipro-job.jp/talent/for-local/top/ にアクセスする。
ログイン後、「案件を探す」または「プロジェクト一覧」へ進む。

### 2. 職種カテゴリで絞り込み
サイドバーまたは絞り込みパネルで以下を選択：

| フィルター項目 | 設定値 |
|-------------|------|
| 職種カテゴリ | config.json の `search.categories` から選択（DX・デジタル化、ITコンサルタント、AI活用 等） |
| 勤務形態 | 「リモート可」「オンライン可」にチェック |
| 地域 | 「全国」または「リモート可」 |
| 稼働形態 | 「副業・兼業」 |

### 3. キーワード検索
検索ボックスに `preferred_keywords` のキーワードを入力（例：「DX」「AI」「業務改善」）

### 4. 案件を収集
- 表示された案件を上から順に確認（最大20件程度）
- COWORK_SKILL.md Step 3 の基準でフィルタリング
- `preferred_keywords` に合致する案件を優先

---

## 応募フォームの埋め方

> ⚠️ 初回アクセス時に実際のフォームを開いて構造を確認し、下記テーブルを更新すること。
> 確認後、このファイルを編集して「確認済み」マークを付けること。

| フィールド | 入力内容 |
|-----------|--------|
| 稼働可能時期 | 最短で稼働可能な日付または「即日可能」 |
| 週稼働時間 | 案件要件に合わせて選択（不明な場合は最小値） |
| 希望単価・報酬 | 「応相談」または案件記載の最低額 |
| 自己PR・応募メッセージ | COWORK_SKILL.md Step 4 のルールで生成した応募文を入力 |
| 添付ファイル | フォームに添付欄があれば `documents/` フォルダ内のファイルを添付 |

---

## 出力ファイル名
`hipro_YYYYMMDD_HHMMSS.jsonl`

## 備考
- ログイン状態が切れている場合は先にログインしてから再実行すること
- フォーム構造が変わっていた場合は、このファイルの「応募フォームの埋め方」を更新すること
