# スキルシフト 応募手順

## サイト情報
- URL: https://www.skill-shift.com
- ログイン: ブラウザで事前にログイン済みであること
- 応募履歴確認: https://www.skill-shift.com/member/application-history

---

## 検索手順

`config.json` の `search_filters` を参照して以下の手順で操作する。

### 1. トップページにアクセス
https://www.skill-shift.com にアクセスする。

### 2. キーワード検索
- トップページまたは案件一覧ページの検索バーに `preferred_keywords` のキーワードを入力して検索
- 複数キーワードは1件ずつ試す（例：「GAS」「AI導入」「業務改善」）

### 3. 絞り込みフィルターを設定
検索結果ページの絞り込みパネルで以下を設定：

| フィルター項目 | 設定値 |
|-------------|------|
| 勤務形態 | 「リモート可」または「完全リモート」にチェック |
| カテゴリ | config.json の `search.categories` から選択（AI・機械学習、業務効率化・自動化、DX・デジタル化 等） |
| ステータス | 「募集中」のみ |

### 4. 案件を収集
- 表示された案件を上から順に確認する（最大20件程度）
- COWORK_SKILL.md Step 3 の基準でフィルタリング
- `preferred_keywords` に合致する案件を優先

## 応募フォームの埋め方

> ⚠️ 初回アクセス時に実際のフォームを開いて構造を確認し、下記テーブルを更新すること。
> 確認後、このファイルを編集して「確認済み」マークを付けること。

| フィールド | 入力内容 |
|-----------|--------|
| 稼働可能時期 | 「すぐに稼働可能」または最短日付を選択 |
| 週稼働時間 | 案件要件に合わせて最小値を選択（不明な場合は「〜10時間/週」） |
| 希望単価・報酬 | 「応相談」または案件記載の最低額 |
| 自己PR・応募メッセージ | COWORK_SKILL.md Step 4 のルールで生成した応募文を入力 |
| 添付ファイル | フォームに添付欄があれば `documents/` フォルダ内のファイルを添付 |

---

## 出力ファイル名
`skillshift_YYYYMMDD_HHMMSS.jsonl`

## 備考
- ログイン状態が切れている場合は先にログインしてから再実行すること
- フォーム構造が変わっていた場合は、このファイルの「応募フォームの埋め方」を更新すること
